# monkey-lang in clojure 
Monkey Lang implement in clojure

## Commands
- ### Run
  ```console
  $ bb run
  ``` 
  or  
  ```console
  $ lein run
  ```  
- ### Test
  ```console
  $ bb test
  ```  
  or  
  ```console
  $ lein test
  ```
